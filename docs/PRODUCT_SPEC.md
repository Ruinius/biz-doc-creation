# Product Specification

## Overview
An Antigravity skill designed to automate and streamline the creation of professional business documents. It allows users to generate high-quality documents such as Proposals, Letters of Intent (LOI), and Statements of Work (SOW), which can then be exported directly into polished Word documents and PDFs.

## Target Audience
- External advisors, independent consultants, and freelancers who frequently draft business proposals and engagement documents.
- The project creator, who built this tool to automate their own proposal writing workflow and is sharing it with the broader consulting community.

## Core Features
- **AI-Powered Document Generation**: An AI agent drafts professional documents using a defined persona (Senior McKinsey Partner) and client-specific context files.
- **Context-Driven Workflow**: Users provide a simple markdown context file with engagement details; the AI handles tone, structure, and formatting automatically.
- **Professional Exporting**: High-quality export to Word (.docx) and PDF formats via Python scripts, maintaining professional styling and layouts.
- **Customizable Skills**: Persona, tone, formatting rules, and document structure are all defined in editable markdown skill files that users can fork and adapt.

## Out of Scope
- Direct email integration or document signing capabilities.
- Complex CRM integrations for the initial phase.
